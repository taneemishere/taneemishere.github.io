Code Demo Paper



Description

LaTeX is a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. It is most often used for medium-to-large technical or scientific documents but it can be used for almost any form of publishing. The main objective of this project is to generate the LaTeX formula code from the images of the formulas. The proposed method is based on the deep learning approach. The proposed method is found to be better than the existing methods.
Combining both computer vision and NLP tasks to generate mathematical formulae from images. The convolutional encoder captures and extracts inner features from images. LSTM based decoder then tries to generate the LaTeX code from the passed token vectors. Trying to achieve the results for a machine which at a time can see and speak, towards generalisation.
