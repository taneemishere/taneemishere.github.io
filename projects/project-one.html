Abstract

Writing HTML code for web pages is a repetitive and time-consuming task for developers to design the structure of a website, they must write the code for each HTML element and different pages of the same structures in websites. A normal person is unable to design the structure of a website because of lack of experience of writing the HTML code. Stakeholders must repeatedly meet the development team for changes of features in website structures to fulfill their organizational needs. To overcome these issues, we will develop an AI based system that will allow developers to get HTML code from both, hand drawn, and computer designed images. This automated system will reduce the development time and omit the repetitive tasks of writing the same HTML code for different web pages. This automated system will also allow a non-technical person to design website structures by providing the images to the system. The stakeholder will not need to meet the development team repeatedly for changes in website structures, they will hand over their organizational needs in the form of hand drawn images to developers and they will generate the HTML code directly from those hand drawn images.
